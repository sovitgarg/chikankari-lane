## Launch checklist

### Setup (one-time)
- [ ] Shopify account created — **DONE** (chikankari-lane-2)
- [ ] Custom App created + API token saved → `config/shopify.env`
- [ ] Razorpay account + KYC submitted
- [ ] Razorpay KYC approved + API keys saved → `config/razorpay.env`
- [ ] Domain purchased + DNS pointed to Shopify
- [ ] Bank account connected in Shopify

### Brand
- [ ] Logo finalized → `assets/logo/`
- [ ] Color palette decided → `brand/colors.md`
- [ ] Typography decided → `brand/fonts.md`
- [ ] Brand voice defined → `brand/voice.md`

### Theme
- [ ] Dawn theme installed
- [ ] Brand colors + fonts applied
- [ ] Homepage sections built
- [ ] Navigation menu configured
- [ ] Footer configured

### Content
- [ ] About page → `content/pages/about.md`
- [ ] The Craft page → `content/pages/the-craft.md`
- [ ] Contact page → `content/pages/contact.md`
- [ ] FAQ page → `content/pages/faq.md`
- [ ] Privacy Policy → `content/policies/privacy.md`
- [ ] Terms of Service → `content/policies/terms.md`
- [ ] Shipping Policy → `content/policies/shipping.md`
- [ ] Returns Policy → `content/policies/returns.md`
- [ ] Refund Policy → `content/policies/refund.md`

### Catalog
- [ ] Product CSV template ready → `catalog/products.csv`
- [ ] Product photos shot + edited → `assets/products/`
- [ ] First batch of products in CSV
- [ ] Collections defined → `specs/collections.md`
- [ ] Smart collection rules configured

### Commerce
- [ ] Razorpay integrated in Shopify
- [ ] COD enabled with min order rules

### Marketing
