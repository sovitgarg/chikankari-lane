# Fonts

`NotoSans-Regular.ttf` and `NotoSans-Bold.ttf` are from Google Noto Sans,
distributed under the SIL Open Font License v1.1.

Source: https://github.com/notofonts/notofonts.github.io
License: https://scripts.sil.org/OFL

Included in this repo because we render PDF reports with Indian rupee (₹, U+20B9)
which the default ReportLab Helvetica font does not contain.
